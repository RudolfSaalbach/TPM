Purpose: Control which external systems can execute which actions, and what parameters they're allowed to use.

  • System: Target system name (e.g., TELEGRAM, N8N, SMTP)
  • Action: Allowed action/command name
  • Parameters: JSON object defining allowed parameter types (e.g., {"message": "string", "priority": "int"})
